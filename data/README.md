# Example dataThis folder contains example datasets so the repository can be run immediately after cloning.## Files* `data_1`* `data_2`* `data_3`## Using a specific datasetThe runner script is `code/run.py`. To switch datasets, edit the `DATA = ...` line in `code/run.py`:```pythonDATA = ROOT / "data" / "data_1"   # or data_2 / data_3```Then run from the project root:```bashpython code/run.py```## NotesThese files are plain-text datasets in the same format expected by `read_data()`. In particular, the loader searches for a header line beginning with `Time`, then reads the time and channel data needed for the pipeline.These files are included for demonstration and testing
